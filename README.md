# Taskly

Система распределения задач с генетическим алгоритмом оптимизации.

## Быстрый старт

### 1. Установить зависимости
```bash
npm install
```

### 2. Настроить базу данных
Создай базу PostgreSQL и выполни:
```bash
psql -U postgres -d taskly -f schema.sql
```

### 3. Настроить переменные окружения
```bash
cp .env.example .env.local
# Заполни DATABASE_URL и JWT_SECRET
```

### 4. Запустить
```bash
npm run dev
```

Открыть: http://localhost:3000

---

## Роли

**Менеджер** — создаёт задачи, запускает оптимизацию, видит все виды плана  
**Работник** — видит задачи, меняет статусы, пишет комментарии

## Виды плана

- **Table** — таблица задач с фильтрами
- **Kanban** — три колонки (To Do / In Progress / Done) с drag & drop
- **Gantt** — временная шкала с полосками задач и нагрузкой команды


## Стек

- **Next.js 14** (App Router)
- **TypeScript**
- **PostgreSQL** (прямые запросы через `pg`)
- **Tailwind CSS**
- **jose** для JWT
- **bcryptjs** для паролей
